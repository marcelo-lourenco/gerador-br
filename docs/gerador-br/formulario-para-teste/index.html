Perguntas Frequentes (FAQ)

Esta página é um 'playground' ao vivo, projetado para ser o alvo de seus scripts de automação de testes. Ela contém uma variedade de campos de formulário que podem ser preenchidos usando as funções do nosso pacote NPM 'gerador-br', permitindo que você pratique e valide suas implementações.

É simples! Escolha a ferramenta de sua preferência (Cypress, Playwright, etc.), clique no ícone para ir ao repositório no GitHub, clone ou baixe o projeto, instale as dependências com `npm install` e execute os testes com `npm test`. Os scripts já estão configurados para interagir com este formulário.

Sim. O pacote 'gerador-br' é uma dependência de desenvolvimento em cada um dos projetos de exemplo. O comando `npm install` cuidará de baixá-lo e instalá-lo automaticamente para você.

Não. O formulário é totalmente 'client-side'. Ao clicar em 'Enviar', um script JavaScript captura os dados preenchidos e os exibe em um modal (pop-up) para que você possa verificar o que foi inserido. Nenhum dado é enviado ou armazenado em nossos servidores.
