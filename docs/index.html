Perguntas Frequentes (FAQ)

Box4Dev (Box for Developers) é uma coleção de ferramentas online e gratuitas projetadas para agilizar o dia a dia de desenvolvedores, testadores e estudantes. Nosso principal produto é o Gerador-BR, uma suíte completa de geração de dados fictícios brasileiros.

Sim, todas as ferramentas disponíveis no Box4Dev são 100% gratuitas para uso. Nosso objetivo é fornecer recursos úteis e acessíveis para a comunidade de desenvolvimento.

É a biblioteca Javascript que alimenta nossas ferramentas online. Você pode instalar o pacote 'gerador-br' em seus próprios projetos Node.js para gerar dados fictícios programaticamente em seus testes automatizados ou para popular bancos de dados de desenvolvimento.

O projeto é open-source! A melhor forma de contribuir é através do nosso repositório no GitHub. Você pode reportar bugs, sugerir novas ferramentas ou até mesmo enviar um pull request com suas próprias melhorias. Toda contribuição é bem-vinda.
